# Contexto del Proyecto para el Asistente IA

## Información General
* **Nombre:** Gestión de Alumnos - Colegio HASPEN
* **Tecnologías:** HTML5, CSS3 (estilos.css) y JavaScript vanila. Desarrollo local sin frameworks.
* **Ubicación:** Desarrollo local en VS Code.

## Estructura Actual del Código
1. **index.html:** Pantalla de Login (Usuario, Contraseña, logo.jpg). Redirige a panel.html.
2. **panel.html:** Panel principal con grilla de 7 módulos:
   - Gestión de Usuarios (vínculo a usuarios.html)
   - Inscripción Alumnos (alerta pendiente)
   - Crear Cursos (alerta pendiente)
   - Materias (alerta pendiente)
   - Calificaciones y Boletín (alerta pendiente)
   - Control de Previas (alerta pendiente)
   - Estadísticas (alerta pendiente)
3. **usuarios.html:** Interfaz del Administrador para dar de alta y listar usuarios. Incluye lógica visual para asignaciones específicas según el rol.
4. **usuarios.js:** Lógica para mostrar/ocultar campos dinámicos y gestionar la acumulación de cátedras en el rol Profesor.
5. **estilos.css:** Estilos unificados para el login, la grilla del panel y el formulario de usuarios.
6. **app.js:** Lógica de validación local y redirección del inicio de sesión.

## Definición de Roles y Permisos (Establecido)
* **Administrador:** Control total. Único que accede a usuarios.html para gestionar cuentas y resetear claves al DNI.
* **Directivo:** Acceso a todo el panel principal para supervisión y estadísticas, excepto al módulo de Usuarios.
* **Preceptor:** Gestión operativa diaria. Se le asignan exactamente 2 cursos de entre los 36 posibles (1º a 6º año, Div. A a F).
* **Profesor:** Perfil pedagógico limitado. Posee una "bolsa de horas" dinámica que le permite tener asignadas múltiples cátedras (Curso + Materia). Va directo a su pantalla de notas al loguearse.

## Reglas de Trabajo
* No generar código por generar; trabajar estrictamente sobre la tarea solicitada.
* Mantener el diseño limpio, institucional y coherente con los estilos CSS existentes.
* El sistema debe permitir licencias docentes (dos usuarios pueden compartir una misma cátedra asignada).
